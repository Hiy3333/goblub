# goblub 사주 2.0 + 운세·오늘의 사주 설계

날짜: 2026-07-03
상태: 승인됨

## 목적

사주 페이지의 풀이 깊이를 전문가 수준 요소(십성·지장간·신강약·진태양시·대운)로 강화하고, 매일 재방문할 이유가 되는 "오늘의 운세"(개인화 4종)와 "오늘의 사주"(공용) 페이지를 추가한다.

## 결정 사항

| 항목 | 결정 |
|------|------|
| 십성 | 일간 기준 10종. 기둥별 천간·지지(본기) 라벨 + 다수 십성 해설 |
| 지장간 | 12지지 테이블. 오행 분포 가중치: 천간 1.0, 지지 본기 1.0, 중기 0.5, 여기 0.3 |
| 신강약 | 일간을 돕는 세력(비겁+인성) 비율. 월지 가중 ×2. ≥55% 신강, ≤45% 신약, 그 외 중화 |
| 진태양시 | 체크박스, 기본 켬. 출생시각 -30분 보정(시주·년월주 경계 판정 모두) |
| 대운 | 성별(남/여) 선택 시 표시. 양남음녀 순행/음남양녀 역행. 대운수 = 출생→절입 일수/3(반올림, 최소 1). 8구간 |
| 운세 페이지 | play/fortune.html. 생일 localStorage('goblub_birth') 저장, 재방문 시 자동 |
| 운세 4종 | 오늘 총운·연애운·재물운(오늘 일진 기준), 이번 달 운세(이번 달 월주 기준) |
| 운세 산식 | 십성 관계 → 카테고리별 기본 텍스트·점수, 내 일지↔오늘 일지 육합/충 → 점수 보정+코멘트. 결정적(랜덤 없음) |
| 오늘의 사주 | play/today.html. 입력 없음. 오늘 년월일 간지 3기둥 + 기운 해설 + 좋은 것/조심할 것 |
| 메인 | 🌞 오늘의 운세, 📅 오늘의 사주 카드 추가 (콘텐츠 6 + 앱 + 빈슬롯 = 8칸) |

## 구조

```
play/js/saju.js      확장: 지장간·십성·신강약·진태양시·대운 + window.Saju API 유지
play/js/fortune.js   신규: 오늘/이번달 간지, 십성→운세 텍스트·점수, 합충 보정, 오늘의 사주 해설
play/saju.html       개편: 성별·진태양시 입력, 십성 라벨, 신강약, 가중 오행 차트, 대운 타임라인
play/fortune.html    신규: 생일 입력(저장) → 운세 4카드
play/today.html      신규: 오늘의 간지·해설(입력 없음)
index.html           카드 2개 추가
```

## 핵심 규칙 (구현 기준)

### 십성

오행 관계 rel = (상대오행 − 내오행 + 5) % 5 → 0 비겁, 1 식상, 2 재성, 3 관성, 4 인성.
음양 same = (상대간 % 2 === 일간 % 2). 이름표:
`[["비견","겁재"],["식신","상관"],["편재","정재"],["편관","정관"],["편인","정인"]][rel][same?0:1]`
지지의 십성은 본기(지장간 마지막 원소) 천간 기준.

### 지장간 (여기→중기→본기 순)

자[임,계]* 축[계,신,기] 인[무,병,갑] 묘[갑,을]* 진[을,계,무] 사[무,경,병] 오[병,기,정] 미[정,을,기] 신[무,임,경] 유[경,신]* 술[신,정,무] 해[무,갑,임]
(*2개 지지는 여기+본기. 가중치는 위치가 아니라 역할: 본기 1.0, 중기 0.5, 여기 0.3)

### 신강약

세력 = Σ(지원 오행 가중치) / Σ(전체 가중치). 지원 = 일간과 같은 오행(비겁) + 일간을 생하는 오행(인성). 월지 항목들은 가중치 ×2. 시주 없으면 있는 글자만으로 계산.

### 대운

- 방향: 년간이 양간(갑병무경임)이고 남자 → 순행. 양남·음녀 순행, 음남·양녀 역행.
- 대운수: 순행 = (다음 절입시각 − 출생시각)의 일수, 역행 = (출생시각 − 직전 절입시각)의 일수. ÷3, 반올림, 최소 1, 최대 10.
- 기둥: 월주에서 순행 +1씩 / 역행 −1씩 60갑자 이동, 8구간. 각 구간 나이 = 대운수 + 10×n.
- 절입시각은 saju-terms.json으로 정확 계산. 연말·연초는 이웃 연도 배열 참조(1900년 이전/2100년 이후로 나가면 대운수는 근사 없이 해당 방향 첫 절만 사용하되, 범위 밖 참조가 필요한 1900년 1월·2100년 12월 출생은 대운수 5로 대체하고 "근사" 표기).

### 진태양시

체크 시 출생시각에서 30분 빼서 모든 판정(년·월주 경계, 일주 23시 규칙, 시주)에 사용. "시 몰라요"면 보정 없음(12:00 가정 유지).

### 운세 산식

- 오늘 일진: (JDN(오늘) − anchorJdn) mod 60. 이번 달 월주: saju.compute의 월주 로직을 오늘 날짜로.
- 기본: 내 일간 vs 대상 천간의 십성 → 카테고리별 텍스트 10종 + 기본 점수(십성별 1~5).
  - 총운 점수: 비견4 겁재2 식신5 상관3 편재4 정재4 편관2 정관4 편인3 정인5
  - 연애운 점수: 비견3 겁재2 식신4 상관4 편재5 정재5 편관3 정관4 편인2 정인3
  - 재물운 점수: 비견2 겁재1 식신4 상관3 편재5 정재5 편관2 정관3 편인3 정인4
  - 이번달 점수표는 총운과 동일, 텍스트만 월 단위 톤.
- 보정: 내 일지↔오늘 일지 육합(자축·인해·묘술·진유·사신·오미) +1 및 코멘트, 충(자오·축미·인신·묘유·진술·사해) −1 및 코멘트. 점수는 1~5로 클램프.
- 사용자 사주는 localStorage 생일로 매번 재계산(시 몰라요 허용).

### 오늘의 사주 해설 조합

- 오늘의 기운 = 일진 천간 오행 텍스트(10종: 천간별) + 지지 특성 텍스트(12종).
- 좋은 것/조심할 것 = 일진 천간의 오행별 5세트(각각 좋은 활동 2~3개, 조심 1~2개).

## 에러 처리

- fortune: 저장된 생일이 손상되면 무시하고 입력 폼 표시. 날짜 유효성은 사주 페이지와 동일 규칙.
- today/fortune: saju-terms.json 로드 실패 시 안내 문구(기존과 동일).
- 오늘 날짜는 사용자 기기 로컬 시간 기준(KST 가정. 해외 접속 보정은 범위 아님).

## 테스트/검증

- 십성: 일간 갑 기준 10천간 전부 기대값 대조(갑=비견, 을=겁재, 병=식신, 정=상관, 무=편재, 기=정재, 경=편관, 신=정관, 임=편인, 계=정인).
- 신강약: 특정 사례 수동 계산 대조 1건.
- 대운: 1995-03-15 14:30 남(을해년=음남 → 역행), 방향·대운수·첫 기둥 수동 대조.
- 진태양시: 23:10 출생 → 보정 시 22:40으로 일주가 안 넘어가는지, 미보정 시 넘어가는지.
- 운세: 같은 입력 반복 호출 = 같은 결과. 육합/충 사례 각 1건 점수 보정 확인.
- 브라우저: fortune 생일 저장 후 새로고침 → 입력 없이 운세 표시. today 페이지 오늘 간지 = saju 일진 계산과 일치.
- 모바일 375px 가로 스크롤 없음. 배포 후 신규 2페이지 + saju 200.
