# 엽기 상황극 MBTI (goblub 에디션) 설계

날짜: 2026-07-03
상태: 승인됨
콘텐츠 원전: `docs/content/wild-mbti-source.md` (Hermes 기획서 — 12문항 JSON·16유형 팩폭/운명/이모지/밈타이틀·계산 로직의 유일한 출처)

## 목적

기존 `play/mbti.html`(초스피드 20문항)을 엽기 상황극 12문항 버전으로 **전면 교체**한다. URL 유지. 콘텐츠는 원전 그대로, 디자인은 goblub 사이트 톤(크림·알록달록·Jua·공용 헤더/푸터)으로 번역한다.

## 결정 사항

| 항목 | 결정 |
|------|------|
| 디자인 톤 | 사이트 톤 통일(다크 네온 아님). 포탈은 팔레트 5색 conic-gradient 회전으로 표현 |
| 기존 테스트 | 완전 교체. quiz.js는 사용하지 않음(전용 인라인 JS). quiz.js 자체는 다른 테스트용으로 유지 |
| 효과음 | 1차 제외 |
| 콘텐츠 수정 | 원문 유지. 단 ISFJ '우주적 운명'은 명칭("산소 나눠주다 질식사")과 어긋나므로 명칭에 맞게 재작성 |
| 계산 | 원전의 calculateMBTI 그대로(축당 3문항×2점, 동점 불가) |

## 화면 흐름 (단일 페이지 내 3상태)

1. **인트로**: 회전 포탈 장식(원형, conic-gradient 5색, CSS 애니메이션, hover 시 가속) + 타이틀 "당신이 만약 미니 식빵이 된다면? 🍞" + 서브 + [🌀 포탈 입장하기] 버튼
2. **진행**: 프로그레스 바(기존 .progress-track 재사용) + 비주얼 카드(문항별 emoji 조합 + tint 배경, 목록은 계획서에 명기) + 질문 텍스트 + A/B 선택 카드 2장(.choice-btn 기반, hover 흔들림 애니메이션) + [📋 이 질문 단톡방에 복사] 버튼(질문+선택지 텍스트 클립보드 복사 → "복사됨!")
3. **결과**: 유형 이모지(대형) + 밈 타이틀 + `유형 (별명)` + 팩폭 문단 + 우주적 운명 문단 + 버튼: [🖼 짤 저장](Canvas PNG 다운로드) [🔗 결과 링크 복사](`URL#유형`) [다시하기]

## 기능 상세

- **해시 진입**: 로드 시 `location.hash`가 유효 MBTI(16종)면 테스트 건너뛰고 결과 즉시 렌더. 무효면 무시.
- **결과 링크 복사**: `location.origin+pathname+"#"+type` 클립보드 복사, 실패 시 URL 텍스트 표시.
- **짤 저장**: Canvas(720×900) — 크림 배경, 팔레트 상단 띠, 유형 이모지(대형 텍스트), 밈 타이틀(줄바꿈 처리), 유형+별명, 하단 "goblub.vercel.app" 워터마크 → PNG 다운로드(`goblub-mbti-<유형>.png`). 외부 라이브러리 없음.
- **질문 복사**: `Q. <질문>\nA. <선택지A>\nB. <선택지B>\n👉 나도 하기: <URL>` 형태.

## 에러 처리

- 클립보드 API 실패(비보안 컨텍스트 등): 복사 대신 텍스트를 화면에 노출.
- Canvas 다운로드는 데스크톱/안드로이드 기준. iOS 사파리는 새 탭 이미지 표시로 폴백(toBlob→open).

## 검증

- 전부 A 선택 → ESTJ, 전부 B 선택 → INFP (원전 배점 기준 기대값) 확인
- 축별 조합 1건 추가(예: A,B 교차) 수동 대조
- `#ENFP` 해시 접속 → 결과 즉시 표시, 무효 해시(#ABCD) → 인트로
- 질문 복사·결과 링크 복사 동작, 짤 PNG 생성(다운로드 트리거) 확인
- 모바일 375px 가로 스크롤 없음, 선택 카드 터치 크기 적절
- index.html 카드 교체(🍞 엽기 상황극 MBTI) 후 이동 확인, 배포 후 실사이트 200
