## 상세

### saju-terms.json 포맷

```json
{
  "anchorJdn": 2460671,
  "note": "anchorJdn = 갑자(甲子)일. terms[year] = [소한,입춘,경칩,청명,입하,망종,소서,입추,백로,한로,입동,대설] 절입시각 'MM-DD HH:MM' (KST)",
  "terms": { "1900": ["01-06 03:04", "02-04 14:51", "..."], "...": [] }
}
```

- 24절기 중 월주 경계인 12절(節)만 포함. 중기(우수·춘분 등)는 제외.
- 년주 경계는 입춘(배열 인덱스 1)을 그대로 사용.

### 사주 계산 규칙 (play/js/saju.js)

- 입력: 양력 생년월일 + 시각(선택). 시각 "몰라요" 선택 시 시주 생략.
- 년주: 입춘 절입시각 이전 출생이면 전년도 간지. 60갑자: (년-4) mod 60 기준.
- 월주: 직전 절(節) 기준 월 결정(입춘=인월…소한=축월). 월간은 년간에 따른 오호둔(五虎遁) 규칙.
- 일주: JDN 계산 후 (jdn - anchorJdn) mod 60 → 60갑자 인덱스.
- 시주: 23시 기준 자시 시작, 2시간 단위 12지지. 시간(時干)은 일간에 따른 오서둔(五鼠遁) 규칙. 23시 이후 출생은 다음날 일간 기준(야자시 논쟁은 단순화: 23시부터 다음날로 처리).
- 오행: 8글자(시주 없으면 6글자)의 천간·지지 각각 오행 매핑, 분포 집계.
- 풀이: 일간 10종(갑을병정무기경신임계)별 성격 텍스트 + 가장 많은/없는 오행 코멘트 + 띠(년지).

### 퀴즈 엔진 (play/js/quiz.js)

- 입력: `{ title, questions: [{q, choices: [{text, scores: {키: 점수}}]}], results: {키: {emoji, name, desc}}, resolve(totals) → 결과키 }`
- 렌더: 시작 화면 → 문항(진행바, 한 문항씩) → 결과 카드(이모지·이름·설명) + 다시하기 + 링크 복사 버튼.
- MBTI는 resolve에서 축별(E/I, S/N, T/F, J/P) 다수결로 4글자 조합.
- 스트레스 테스트 결과 화면에만 Golden time 앱 안내 한 줄 + apps.html 링크.

### 페이지 공통

- 각 페이지는 기존 규칙(`data-root=".."`, 공용 헤더/푸터, css/style.css)을 따른다.
- index.html 카드: 🔮 사주(tint-purple), 💘 연애 스타일(tint-coral), 🔥 스트레스(tint-yellow), 🎭 MBTI(tint-mint). "새 즐길거리" 빈 슬롯은 그리드 맨 뒤로 이동.
