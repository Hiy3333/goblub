# [기획서] 엽기 상황극 MBTI 테스트 (Wild Scenario MBTI Test)
*작성일자: 2026-07-03 | 기획 파트너: Hermes*

---

## 1. 서비스 컨셉 및 핵심 가치
* **서비스명**: 엽기 상황극 MBTI 테스트 (Wild Scenario MBTI)
* **한 줄 컨셉**: 지루하고 이탈률 높은 뻔한 성격 질문 대신, **초극단적이고 기괴한 12가지 상황극**을 제시하여 유저의 '동물적 본능'을 통해 MBTI를 판별하는 고도화된 스낵 콘텐츠 앱.
* **디자인 시스템**: 네온 레트로 & 픽셀 다크 모드 (어둡고 감각적인 우주/사이버 톤배경에 톡톡 튀는 비주얼 에셋 배치).
* **개발 지향점**: 본 기획서는 Claude Code 및 프론트엔드 개발자가 **즉각적인 JSON 파일 로드 및 UI 구현**을 할 수 있도록 완벽한 사양을 포함함.
* **배포 형태**: **의존성 없는 단일 HTML 파일 1개** (CSS/JS 인라인, 외부 이미지·폰트·라이브러리 없음). 캐릭터/배경 비주얼은 이미지 파일 대신 **이모지 + CSS 픽셀아트/그라데이션**으로 표현하고, 효과음은 Web Audio API로 합성(음소거 토글 제공).

## 2. 화면 구성 및 사용자 흐름 (UI/UX Wireframe)

### 🖥️ Page 1: 메인 대문 화면 (Intro)
* **비주얼 에셋**: `[CHAR_HERO]` (눈이 핑핑 도는 캐릭터 일러스트), `[BG_SPACE_PORTAL]` (우주의 시공간 소용돌이 배경).
* **타이틀**: "당신이 만약 미니 식빵이 된다면? 🍞" 
* **서브 타이틀**: "뻔한 착한 척 설문지는 가라. 우주적 대위기 상황에서 펼쳐지는 당신의 찐 성격 본능 테스트."
* **마이크로 인터랙션**: 
  * '우주 시공간 포탈 입장하기' 버튼 마우스 오버 시 포탈이 세차게 회전하는 CSS 애니메이션.
  * 입장 클릭 시 기괴하고 중독성 있는 효과음과 함께 화면이 빨려 들어가는 트랜지션 연출.

### 📱 Page 2: 테스트 진행 화면 (Test Play)
* **Layout**: **'에셋 매핑 카드 타임라인 뷰 (Variant A)'**
  * 화면 상단: 프로그레스 바 (1/12) 및 현재 진척도 표시.
  * 화면 중앙: 현재 질문의 배경 `[BG_TOASTER]`, 상황 캐릭터 `[CHAR_BREAD]`·소품 `[ITEM_BUTTER]` 에셋 카드가 입체적인 일러스트 형태로 시각화됨.
  * 화면 하단: 기괴하고 매력적인 상황극 질문 텍스트와 **직관적인 2개의 선택지 카드 (A/B)**.
* **마이크로 인터랙션**:
  * 선택지 카드에 마우스를 올리면 픽셀 캐릭터가 살짝 떨리거나 반응하는 모션.
  * **'질문 복사' 플로팅 아이콘**: "이 질문 미쳤다" 싶은 개별 질문을 단톡방에 복사해 기습 질문할 수 있는 **원클릭 복사 마이크로 인터랙션** 장착.

### 🏆 Page 3: 결과 화면 (Result Report)
* **비주얼 에셋**: 판정된 MBTI 결과 캐릭터 일러스트(예: `[CHAR_VILLAIN_INFP]` - 우주 먼지 몽상가)가 중앙에 배치됨.
* **구성 요소**:
  * **골 때리는 밈 타이틀**: (예: "토스터기에서 불탄 재가 된 우주 먼지 몽상가 (INFP)")
  * **성격 팩폭 분석**: 장황한 장점 나열 대신, 엽기 상황극 선택들을 종합해 "당신의 생존 확률과 찌질함"을 팩트 폭격하는 위트 있는 분석 리포트.
  * **원클릭 짤방 다운로드**: 내 결과를 힙한 신문 속보나 게임 클리어 픽셀 카드 형태로 내려받거나 클립보드에 복사하는 버튼. (Canvas API로 결과 카드를 그려 PNG 저장 — 외부 라이브러리 불필요)
  * **결과 링크 공유**: URL 해시에 결과 유형을 담아(`#INFP`) 클립보드에 복사하는 버튼. 해당 링크로 접속하면 테스트를 건너뛰고 결과 화면이 바로 열림.
  * **다시하기 버튼**: 첫 화면으로 돌아가 재도전. (친구 유형 대리 테스트 유도 — 바이럴 핵심 동선)

## 3. 비주얼 에셋 일관성 토큰 테이블 (Asset Token Tracking)
개발 시 비주얼 일관성을 위해 캐릭터, 배경, 소품을 아래의 고유 토큰으로 규정하여 일관되게 트래킹합니다. 단일 HTML 구현에서는 이미지 파일 대신 **'HTML 구현' 컬럼의 이모지/CSS 표현**을 사용하고, 추후 일러스트 에셋 제작 시 '프롬프트 스타일 가이드'를 활용합니다.

| 토큰 ID | 에셋 분류 | 설명 | HTML 구현 (이모지/CSS) | 프롬프트 스타일 가이드 |
| :--- | :--- | :--- | :--- | :--- |
| `[BG_SPACE_PORTAL]` | 배경 | 시공간이 일그러진 퍼플-블랙 소용돌이 포탈 | CSS conic-gradient 회전 애니메이션 (퍼플-블랙) | Neon purple space portal, pixel art, retro sci-fi |
| `[BG_TOASTER]` | 배경 | 붉은 열선이 지글거리는 메탈릭 토스터기 내부 | CSS 다크 메탈 그라데이션 + 붉은 glow 라인 | Inside of a glowing red metallic toaster, pixel art |
| `[CHAR_HERO]` | 캐릭터 | 눈동자가 뱅글뱅글 도는 귀엽고 하찮은 픽셀 인간 | 😵‍💫 (대형 이모지 + 네온 테두리) | Derp eyes pixel character, neon green outlines, cute |
| `[CHAR_BREAD]` | 캐릭터 | 토스터기 속 절망에 빠진 귀여운 10cm 미니 식빵 | 🍞 + 😱 조합 | Distressed mini sliced bread with face, pixel art |
| `[ITEM_BUTTER]` | 소품 | 식빵 캐릭터 위에 올려진 살짝 녹아가는 버터 조각 | 🧈 | Melting butter chunk on top, retro 8-bit asset |

각 질문 화면의 배경/이모지 조합은 질문 JSON의 `visual` 필드로 매핑합니다 (예: `"visual": { "bg": "space", "emoji": "🛸📻" }`). 16유형 결과 캐릭터 `[CHAR_VILLAIN_XXXX]`의 이모지 매핑은 5번 섹션 각 유형 항목에 명기.

### 1. INFP (우주 먼지 몽상가) | `[CHAR_VILLAIN_INFP]`
* **명칭**: 토스터기에서 불탄 재가 된 우주 먼지 몽상가
* **성격 팩폭**: 자존감은 지구 핵 내핵에 처박혀 있는데 상상력은 안드로메다 우주 정거장급. 우주 멸망 소식이 들리면 생존 도구를 챙기는 대신 "마지막 날을 장식할 감성 일기장"을 사러 가며, 정작 대처법은 침대 이불 밑에서 베개 안고 고민함.
* **우주적 운명**: 토스터기 안에서 아름답게 구워지는 완벽한 버터 식빵이 되는 몽상만 가득 품다가, 탈출 시도조차 안 한 채 결국 석탄처럼 새까맣게 불타 흩어지는 먼지가 됨.

### 2. INFJ (파멸의 심연 예언자) | `[CHAR_VILLAIN_INFJ]`
* **명칭**: 외계인과 텔레파시 시도하다 지구가 멸망한 은둔 교주
* **성격 팩폭**: 도통 속을 알 수 없는 기괴한 인류애와 은밀한 파괴욕의 기묘한 하이브리드. 혼자 머릿속으로 지구를 세 번 구하고 다섯 번 멸망시키는 어둠의 시나리오를 집필함.
* **우주적 운명**: 우주선 산소가 부족할 때 울고 짜는 동료를 우수에 찬 눈빛으로 쳐다보며 "이것도 기나긴 우주의 정화 섭리야..." 하고 혼자 해탈해서 동료 몰래 산소 밸브를 아주 차분히 줄여버림.

### 3. INTP (분석 광기 인공지능) | `[CHAR_VILLAIN_INTP]`

* **성격 팩폭**: 행동력은 우주 나무늘보 수준인데, 잘난 척하고 싶은 지적 호기심은 무한대. 남들 울고 짜며 좀비에 물어뜯길 때 혼자 구석에서 "좀비의 타액 내 바이러스 전파 원리"를 메모하며 눈을 빛냄. 귀찮아서 그냥 좀비가 다가와도 목을 대줌.
* **우주적 운명**: 미니 식빵이 되었을 때 탈출할 생각은 안 하고, 식빵 밀가루 성분이 열을 받아 탄화되는 9단계 화학 과정을 머릿속으로 논문 작성하다 타버림.

### 4. INTJ (어둠의 우주 정복자) | `[CHAR_VILLAIN_INTJ]`
* **명칭**: 지구 멸망 대비 방공호 짓다 좀비에게 열쇠 뺏긴 유리몸 설계자
* **성격 팩폭**: 머릿속으로는 플랜 Z까지 세워두어 이미 지구를 3번 정복했으나, 정작 실제 방문을 여는 데 필요한 악력과 물리적 근력이 부족함. 고집불통 독고다이의 대명사.
* **우주적 운명**: 좀비 아포칼립스 생존 매뉴얼을 5년간 완벽하게 컴퓨터에 짜놓았지만, 좀비 습격 첫날 대형 마트 카트의 고장 난 바퀴에 발가락이 끼어 무릎을 꿇은 채 끔살당함.

### 14. ESFJ (평판 광기 참견 헌터) | `[CHAR_VILLAIN_ESFJ]`
* **명칭**: 지구 멸망 10분 전인데 3대 단톡방에 작별 인사 안 남겼다고 친구 차단한 소외감 빌런
* **성격 팩폭**: 모든 친목 모임의 인맥 왕이자 완벽한 평판 주의자. 남들의 가십과 뒷담화를 온 은하계에서 가장 빠르고 세련되게 전달하며, 겉으로는 누구보다 세상에 친절한 마더 테레사를 완벽하게 연기함.
* **우주적 운명**: 엘리베이터 정전 속에서 나와 완벽하게 똑같이 생긴 도플갱어를 만나자마자 "어머 자기 피부 너무 좋다! 톤업 크림 어디 거 써?" 하며 수다를 떨다 포탈 탈출 시간을 전면 오버함.

### 15. ESTP (우주 파괴 폭주 오토바이) | `[CHAR_VILLAIN_ESTP]`
* **명칭**: 영혼 자판기 부수고 음료 몽땅 털어 훔쳐 달아난 사기 충동의 액션 빌런
* **성격 팩폭**: 뇌를 거치지 않는 직발성 주먹과 입방정. 일단 들이받고 수습은 평행우주 저 너머의 다음 차례 내가 하게 둠. 성격이 불도저 같아서 오히려 지가 당해놓고 뻔뻔하게 배를 내미는 무시무시한 야생동물.
* **우주적 운명**: 비행기 폭우 결항 사태에 활주로 철조망을 뜯고 들어가 "오히려 좋아! 오늘 밤은 비행기 날개 위에서 조개구이 구우며 차박이다!"를 외치며 난동 부리다 공항 특수 진압 부대에 제압당함.

### 16. ESTJ (우주 꼰대 잔소리 폭격기) | `[CHAR_VILLAIN_ESTJ]`
* **명칭**: 지구 멸망 충돌 3초 전인데 유언장에 맞춤법 틀렸다며 친구 따귀 때린 빨간 펜 지휘관
* **성격 팩폭**: 타인의 슬픔이나 아픔에 대한 공감 능력은 진공 상태인데, 잔소리와 지배욕은 은하 규모. 내 말이 무조건 정답이며, 내 맘에 안 들면 세상의 모든 기획서와 사람의 성격을 쓰레기 취합하듯 난장판을 침.
* **우주적 운명**: 상사의 개소리 피드백으로 내 기획서가 까인 날, 사직서 대신 그 상사가 사내 클라우드에 몰래 올린 맞춤법 틀린 엉망진창 공금 영수증 청구서 엑셀 파일을 해킹해서 전사 단톡방에 박제해 복수함.

## 6. 최종 MBTI 결과 연산 로직 (Developer Spec)
프론트엔드 개발 시 누적된 `E, I, S, N, T, F, J, P` 스코어를 기반으로 최종 문자열을 도출하는 자바스크립트 로직 예시입니다.

```javascript
function calculateMBTI(scores) {
  let mbti = "";
  mbti += scores.E >= scores.I ? "E" : "I";
  mbti += scores.S >= scores.N ? "S" : "N";
  mbti += scores.T >= scores.F ? "T" : "F";
  mbti += scores.J >= scores.P ? "J" : "P";
  return mbti;
}
```
*동점 처리 참고*: 축당 3문항 × 2점 올인 배점이므로 축별 점수는 6:0, 4:2, 2:4, 0:6만 가능하며 **동점은 발생하지 않음**. 위 `>=` 비교는 방어적 디폴트(E/S/T/J 우선)로만 유지.

*결과 링크 진입 처리*: URL 해시에 유효한 MBTI 문자열(예: `#ENFP`)이 있으면 테스트를 건너뛰고 해당 결과 화면을 즉시 렌더링. 유효하지 않으면 무시하고 인트로 표시.
